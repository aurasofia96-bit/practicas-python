# 🐍 Mis Prácticas de Python

## 📌 ¿De qué trata esto?
Este repositorio es como mi cuaderno de notas pero en código. Aquí voy subiendo todos los ejercicios que hago para la universidad, los retos de lógica que voy resolviendo y los pequeños programas que invento para soltar la mano con **Python**.

Lo tengo público para ver mi propio progreso y tener todos mis scripts ordenados en un solo lugar.

---

## 📂 ¿Qué tipo de cosas hay aquí?

Como este repositorio va a ir creciendo, los archivos están organizados para resolver diferentes tipos de problemas:

* **🧮 Operaciones y Matemáticas:** Ejercicios con números, fórmulas y lógica (como calcular factoriales o jugar con porcentajes).
* **📅 Fechas y Cosas Aleatorias:** Scripts que usan el tiempo del computador o que eligen opciones al azar (como contar días para un evento o elegir mensajes).
* **🖥️ Programas por Consola:** Retos interactivos donde la terminal te pide datos, te responde y se asegura de que no metas una opción incorrecta.
* **📦 Listas y Módulos:** Ejercicios para guardar datos organizados y conectar un archivo de Python con otro.

---
